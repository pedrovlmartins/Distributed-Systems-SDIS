Para correr este programa primeiro terá que o compilar:
Corra o ficheiro bash "compile.sh" introduzindo o comando "sh compile.sh" na linha de comandos.

Deve agora abrir uma linha de comandos no diretório "src".

Depois deste passo poderá iniciar um peer executando o seguinte comando:

java peers.Peer <Server ID> <MC> <MC port> <MDB> <MDB port> <MDR> <MDR port>

Poderá executar a TestApp executando o seguinte comando:

java launcher.TestApp <peer_ap> <sub_protocol> <opnd_1> <opnd_2>
